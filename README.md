# job_hunter.py

An AI-powered job hunting toolkit built with Python and the Anthropic Claude API. Designed for professionals navigating career transitions who want to stop rewriting the same resume over and over.

---

## What It Does

Paste a job description. Get a tailored resume, cover letter, and interview prep — all generated by Claude in seconds.

**Five tools. One command.**

| Tool | What It Does |
|------|--------------|
| Job Relevance Scorer | Rates how well a job matches your background. Gives you a score, match reasons, gaps, and a verdict: APPLY / SKIP / MAYBE |
| Resume Tailor | Rewrites your resume to mirror the job description language, weighting bullets proportionally to the role's responsibilities |
| Cover Letter Generator | Generates a confident, 3-paragraph cover letter in your voice — no "I am writing to apply" openers |
| Interview Prep | Surfaces the top 5 likely questions, suggested answer frameworks, smart questions to ask, and a key thing to emphasize |
| Full Pipeline | Runs all four tools in sequence for a complete job application package |

---

## How It Works

1. Save a job description to `job.txt`
2. Run `python3 job_hunter.py`
3. Choose a tool from the menu
4. Results are printed to the terminal and saved to `job_results.md`

```
=== JOB HUNTER TOOLKIT ===
1. Score a job listing
2. Tailor my resume
3. Generate a cover letter
4. Interview prep
5. Full pipeline (score + tailor + cover letter + prep)
6. Resume + cover letter only

Choose an option (1-6):
```

## Setup

### Requirements

- Python 3.8+
- An Anthropic API key

### Install dependencies

```bash
pip install langchain-anthropic langchain-core python-dotenv
```

### Add your API key

Create a `.env` file in the project root:

```
ANTHROPIC_API_KEY=your_key_here
```

### Configure your background

Open `job_hunter.py` and fill in:

- `YOUR_BACKGROUND` — your skills, experience, tools, and career goals
- `BASE_RESUME` — your standard resume in plain text format
- `EXPERIENCE_BANK` — a list of strong bullets tagged by employer, used as a fact base for generating tailored content

The more detail you provide, the better the output.

## Usage

```bash
# Save a job description first
cat > job.txt
# Paste the job description, then press Control+D

# Run the toolkit
python3 job_hunter.py
```
